#Scan-N-Dine:The Automated Canteen Management System.
## 📌 Project Overview

Scan-N-Dine is a web-based canteen management system designed to eliminate long queues and manual ordering. Customers simply scan a QR code placed on their table to access the digital menu and place orders directly from their mobile devices.

The system also provides an admin panel for menu management, order tracking, and order status updates.

---
 ✨ Features

### 👨‍🍳 User Features

- QR Code based menu access
- Browse menu by category
- Search food items
- Attractive food catalog
- Add items to cart
- Update item quantity
- Apply discount/coupon codes
- Table number selection
- UPI payment integration (UI)
- Digital order receipt
- Responsive and modern UI
- Limited-time offer coupons

---

### 🔐 Admin Features

- Secure admin login
- Add new food items
- Update existing menu items
- Delete menu items
- Manage food categories
- Upload food images
- Enable/Disable menu items
- View all customer orders
- Search orders
- Filter orders by status
- Mark orders as:
  - Pending
  - Completed
  - Cancelled

---

## 📱 User Workflow

1. Open the homepage.
2. Select **User Order**.
3. Scan the table QR code.
4. Browse food categories.
5. Search for desired food.
6. Add items to cart.
7. Apply available coupon code.
8. Enter table number.
9. Proceed to checkout.
10. Complete payment using UPI.
11. Receive digital order receipt.
